Comandi base

Qua verrano mostrati i comandi principali

Navigazione tra cartelle

Per aprire Windows Terminal è possibile cercare "Windows Terminal" nel menu Start oppure premere Win + X e selezionarlo dal menu.

Comando pwd

In PowerShell:

pwd

Mostra il percorso della cartella corrente.

Esempio

Output:

C:\Users\Studente\Documents

Comando dir

In PowerShell:

dir

Visualizza il contenuto della cartella corrente.

Esempio

Output:

Documenti Download Immagini Desktop

Comando cd

In PowerShell:

cd

Permette di spostarsi tra le cartelle.

Come usarlo:

cd nome_cartella

Comando per tornare alla cartella precedente

In PowerShell:

cd ..

Tornare alla cartella precedente.

Creazione e gestione di Elementi

Comando mkdir

In PowerShell:

mkdir

Crea una nuova cartella

Esempio

mkdir Repo_esame

Comando echo

In PowerShell:

echo

Creare un file di testo

Esempio

echo Ciao > file.txt

Viene creato il file file.txt contenente la parola "Ciao".
